Detailed Description

Performs Ruiz scaling on the optimization problem data.

This function implements the Ruiz equilibration algorithm to improve the numerical conditioning of the optimization problem. It iteratively scales the rows and columns of the constraint matrix A and adjusts the corresponding vectors accordingly.
