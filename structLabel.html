Detailed Description

Represents a label used in a solver.

This struct contains various properties and methods related to a label used in a solver. It stores information such as the set of Fj, id, is_extended flag, vertex, cost, real_cost, SRC_cost, resources, predecessor, is_dominated flag, nodes_covered, nodes_ordered, node_id, cut_storage, parent, children, status, visited, and SRCmap.

The struct provides constructors to initialize the label with or without a node_id. It also provides methods to set the covered nodes, add a node to the covered nodes, check if a node is already covered, and initialize the label with new values.

The struct overloads the equality and greater than operators for comparison.
