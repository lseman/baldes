Detailed Description

A sparse matrix in Compressed Row Storage (CRS) format.

The sparse matrix stores non-zero elements and allows efficient row-wise access through iterators. The SparseMatrix struct also provides methods to insert elements, build the CRS structure, and convert the matrix to a dense representation.

Build the CRS structure by computing row start positions.

This function sorts the non-zero elements by row and column and then computes the starting index for each row in the elements vector. This enables efficient row-wise traversal.
