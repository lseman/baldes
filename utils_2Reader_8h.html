inline

Reduces the time windows for the vertices in the VRPTW instance.

This function iteratively reduces the time windows for the vertices in the VRPTW (Vehicle Routing Problem with Time Windows) instance. The reduction is performed based on the distances, service times, and existing time windows of the vertices. The function stops iterating if no further reduction is possible or after a maximum number of iterations.
