Detailed Description

Sparse matrix in CRS format with optimized row-wise iteration.

This file defines a SparseMatrix struct that represents a sparse matrix in Compressed Row Storage (CRS) format. It supports efficient row-wise access to non-zero elements and can convert the sparse matrix to a dense format.

The struct uses an optimized build function to convert a list of sparse elements into CRS format, enabling fast row-wise traversal using an iterator.
