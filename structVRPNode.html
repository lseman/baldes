Detailed Description

Represents a node in a Vehicle Routing Problem.

This struct contains information about a node, such as its ID, start time, end time, duration, cost, demand, and capacity constraints. It provides constructors to initialize the node with different sets of parameters. The setDuals method allows updating the cost of the node.

Compute the strongly connected components (SCCs) for a VRP graph.

Given a vector of VRPNodes, this function builds a graph using each node's forward arcs (assumed to represent directed edges from node.id to arc.to), and runs Tarjan's algorithm to compute the SCCs.
