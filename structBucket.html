Detailed Description

Represents a bucket.

A bucket is a data structure that contains labels, node ID, lower bounds, upper bounds, forward arcs, backward arcs, forward jump arcs, and backward jump arcs. It provides methods to add arcs, add jump arcs, get arcs, get jump arcs, add labels, remove labels, get labels, clear labels, reset labels, and clear arcs.
